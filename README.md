# Video Editor

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/SlavikLutsenko/test_video_editor.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or
   ```sh
   yarn
   ```

## Running Project

- To start the local server, use the following command:
  ```sh
   npm run start
  ```
  or
  ```sh
   yarn run start
  ```
